Aide pour l'utilisation

  • Les fonctionnes fondamentales de l'application sont presque évidentes. Au contraire, les parties suivantes de son comportement ont besoin d'explication :

    Modification du système de fichiers

    Glissez votre doigt sur une fichier ou dossier. Une « fiche d'action » apparaîtra. Ici, vous devrez faire les opérations plus importantes : copier, couper, supprimer, etc.

    Si vous touchez le bouton à droite, une page sera présentée, qui va vous permettre de modifier les propriétés du fichier ou dossier, ou le compresser (en plus, vous pouvez ajouter les images et les vidéos à l'album de photos et les musiques à l'iPod). Quand vous avez fini, appuyer « OK » si vous voulez enregistrer les changements, ou touchez « fermer » si vous ne le voulez pas.

    Modification d'images

    Pendent le visionnement d'une image, appuyer le bouton à droite. Vous pouvez choisir « Effets de l'image » et les appliquer sur l'image. Apres vous avez fini, choisissez « Enregistrer » si vous voulez enregistrer les changements.

    Ajouter morceaux à l'iPod

    Alerte ! Cette fonction est encore éxpérimentale. Après l'utiliser, iPod ne verra pas les chansons et les vidéos. Au contraire, MyFile (MyPod) les verra, et après une synchronisation avec iTunes ou une application utilisant libgpod, tout sera bien encore. Avant vous pouvez ajouter morceaux à l'iPod, vous devrez visiter ce lien: http://ihash.marcansoft.com/ et créer un fichier nommé HashInfo en utilisant le UDID de votre appareil et le mettre dans /var/mobile/Media/iTunes_Control/Device. Après synchroniser avec une application utilisant libgpod ou iTunes.

    Installation de paquets

    MyFile peut installer les paquets DEB et IPA. Pour le dernier, il utilise le script BASH installApp (/usr/libexec/installipa). Cette application est écrite par les développeurs de Installous.

Changelog for MyFile v1.5 (bugfix and feature release)
    • V hlavním paneluse nyní zobrazuje počet souborů a volného místa v aktuálním adresáři
    • Přidal Vytáhněte si pro obnovení v hlavním zobrazení, Dropbox panel a MyPod
    • Přidána funkce pro přidávání obrázků a videa do fotoalba
    • IPA instalace
    • Přidána anglická, francouzká,česká a maďarská lokalizace
    • Přidán iPod integrace! Vlastnosti: přidat skladby a videa do iPodu knihovny, přehrávat písně a videa z iPodu knihovny ( Akce - > MyPod ) - Experimentální: nefunguje tak, jak by měl / vyčkejte, přečtěte si nápovědu!
    • Přidal jednoduchý grafický editor na prohlížení obrázků ( rotace, škálování / změnu velikosti, mirror / převrátit, oříznout / střih, efekty ( zaobleným rohů, alpha, stupně šedi ), atd...)
    • Odstraněna tlačítka Předchozí / Následující z Image Viewer: používat gesta ( snímek vlevo / vpravo na předchozí / následující dvěma prsty, repectively )
    • Provedení příkazu: na konci textu je vždy prezentován a aplikace zastaví, když uživatel zavře okno.
    • Sdílet na Twitteru: Můžete potvrdit, jestli opravdu chcete sdílet soubory a můžete také změnit text zprávy.
    • Chyba opravena: paket manažer nemůže rozbalit soubory s názvy obsahující mezery.
    • Opravena chyba v prohlížeči obrázků, který velikost snímku "zdědila" z předchozího obrazu a tím byl zkreslený
    • Audio přehrávač: místo mého provedení je UIImagePickerController používá k výběru image album
    • Opravy uživatelské rozhraní
Funkce
    • Administrátorský přístup, podpora celý souborový systém
    • Tvorba, mazání, přejmenování, přesun, kopírování souborů a adresářů
    • Komprese adresářů a souborů do komprimovaného tar, bzipped tar, zip, XAR a deb souboru
    • Prohlížení a editaci různých typů souborů, včetně,textového souboru ( TXT, plist, struny, C, CPP, m, mm, JS, CSS, HTML, HTM, PHP, XML,...), HTML stránek (HTML, htm ), image (jpg, gif, png, bmp...), SQL datového souboru ( SQLite, SQL, DB, sqlitedb, BTI ), Apple List vlastnictví ( plist, smyčce ), zvukové soubory ( MP3, M4A, M4R, WMA...), video souborů ( MP4, MOV,...), binárních ( o, dylib, bin,... ) ( pomocí hex editor), komprimované soubory a balíčků (zip, tar, gz, tar.gz, tar.bz2, deb, IPA, rar) a tak dále.
    • Rozpoznat,označit a řešení symlinky
    • Ukázat© AppStore názvy aplikací, nainstalované v ~ / Applications
    • Prohlížení a úpravy některých vlastností souboru, včetně jména, MIME typ ( readonly ), MD5 hash ( readonly ), velikost souboru ( readonly ), Unix CHMOD souboru režimu, vlastníka souboru, skupiny souborů, zvuku a obrazu metadata
    • Plnohodnotné integrace Dropbox:
      • Stahování souborů
      • Nahrávání soubory libovolného typu (nejen fotografie a videa ), nahrání adresáře
      • Správa vzdálených souborů: mazání a přístup k veřejným odkazům ( využívá bit.ly URL služby zkrácení)
      • Vytvoření nové složky
      • Synchronizace lokální složky s Dropbox
    • Integraci společných sociálních služeb, jako je Facebook a Twitter
    • HTTP sdílení souborů přes lokální Wi -Fi sítě - stahování souborů z libovolného místa, nahrávání souborů ( PHP) do ~ / dokumenty ( nahrávání: Návštěva [ iDevice vaše IP adresa ] / var / mobile / upload.html )
    • SFTP sdílení souborů
    • Ochrana heslem pro spuštění aplikace
    • Koš na bezpečnější výmaz
    • Posílání obrázků jako přílohu e-mailem, sdílet na Twitter a Facebook
    • Instalace a odebírání balíčků IPA aplikace a DEB balíčky Debianu
    • Přidání záložky pro soubory a adresáře
    • Podpora Open in: Funkce od Mobile Safari přes mf: / / nebo myfile: / / schema
    • Čtení a zápis do iPod knihovny ( experimentální): Přehrávání skladeb a přidáním
    • Úpravy obrázků ( měřítko, otočení, přidat textové popisky, nastavit průhlednost atd...)
    • Prohlížení a editaci zvuku ( ID3 ) a obraz ( EXIF ) metadata
    • Přidávání obrázků a videa do knihovny fotografií
    • Překlady: angličtina, francouzština, maďarština,čeština
    • A konečně: myfile má být alternativou pro volný správců souborů v IOS. Jako takový, slibuji, že to nebude nikdy placený. Jen jsem vás chtěl požádat o dar, abych mohl pokračovat ve vývoji s novější Apple zařízení. Věřím, ve svobodný softwar.

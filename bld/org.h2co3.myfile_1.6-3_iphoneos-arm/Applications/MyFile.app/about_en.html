  • MyFile makes use of the free Mongoose HTTP server by Valenok and Antirez.

  • MyFile uses the free and opensource Dropbox API and Objective-C SDK

  • MyFile uses a forked version of MGTwitterEngine, originally developed by Matt Gemmell. Thank you, Mr Gemmell! MyFile also uses SA_OAuthTwitterEngine by Ben Gottlieb. Thank you, Mr Gottlieb!
